# DDD - Dotfiles Deployment Directive

This is a program initially to mark the completion of my journey in LearnCpp.com with functionality to quickly deploy user configuration from remote storage to local storage using json interface as the directory management. It uses nlohmann/json .hpp for parsing the json files, a simple std::vector as a container for the conversion of json object and uses std::unordered_map for the interface to retrieve and deploy the configuration files. The code is around 5% written by AI / more or less a total of 12 lines is written by ai to deal with the map read.

## Trivia
The name "DDD" is a satirical reference to "DDD" in the game Honkai: Star Rail, which is an abbreviation for Dance, Dance, Dance!!!.
